Pathfinding-Advanture

The Treasure Hunt game was developed using Python programming language and the Tiled software for designing the map. The game incorporates concepts from graph theory, including the BFS algorithm to find the shortest path from the starting point to the treasure. The game is played on a directed graph with unweighted edges and new random subgraphs are generated each time the game is played. The game also includes various types of graphs such as the wheel graph, complete graph, bipartite graph, bridge, and cut vertex. Players must navigate the map by colliding with pillars to receive questions and keys to unlock the next checkpoint. Each correct answer awards 100 points, and incorrect answers result in a deduction of 50 points. The game ends after five incorrect answers. Overall, the project demonstrates the application of graph theory in game development and highlights the use of Python and Tiled software for game design.
